## 💖 Descripción 

**NANO-MD** es un bot de WhatsApp enfocado en velocidad, estilo y funciones void. Basado en `baileys` oficial, sin mods raros. Diseñado pa grupos, economía, gacha y todo lo que se te ocurra.

**Nombre clave:** `Bocchi System` | Código ligero, sin bloat, optimizado pa Termux y VPS.

---

## 💫 Características Void

- **Sistema Gacha**: Invoca personajes, guarda tu colección
- **Economía Void**: Banco, robar, trabajar, casino
- **Gestión de Grupos**: Anti-link, welcome, detect, warn
- **IA Integrada**: Chat con Gemini, genera stickers con texto
- **Descargas**: YouTube, TikTok, Instagram sin API
- **Juegos**: TicTacToe, Ahorcado, Trivia
- **Tools**: SS Web, HD, Removebg, Traductor
- **100% Custom**: Cambia nombre, prefijo, todo

## 💞 Información Importante

1. Usa solo la librería **oficial de Baileys**. Nada de forks o "baileys-mod"
2. No spam. WhatsApp banea si abusas

pm2 save

//títulos
